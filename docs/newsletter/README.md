# Newsletter Archive

This directory stores the daily Long Tail Inference Lab digest.

The digest is generated by `scripts/build_daily_newsletter.py` and committed by the scheduled GitHub Actions workflow. It tracks public signals related to edge LLM inference, open weight model serving, local runtimes, laptop plus VPS split inference, network tax measurement, job level routing, quantized models, and verified everyday compute experiments.

Run it locally:

```bash
python scripts/build_daily_newsletter.py
```

The generated files are:

- `YYYY-MM-DD.md` for the dated digest.
- `latest.md` for the newest digest snapshot.
