## Variables and controls

## Experiment sequence

### Phase 1: baseline

Describe the baseline and the exact measurements.

### Phase 2: intervention

Describe the system or configuration being evaluated.

### Phase 3: analysis

Describe comparisons, uncertainty, and failure analysis.

## Metrics

Define every metric and its unit. Avoid ambiguous terms such as faster, cheaper, or better without a calculation.

## Reproduce

List the evidence required before the experiment can be marked Complete.

## Results

## Limitations
