# Resources

Resources are reusable materials that support many experiments.

They are not active projects and they do not have a completion date.

## Learning

The learning collection provides a structured path through inference systems, local models, distributed execution, routing, memory, and session state.

[Open the field guide](learning/field_guide.md)

## Project proposals

Project proposals describe larger systems directions that may later produce several experiments.

### [Privacy Aware Inference Boundary](project_proposals/privacy_aware_inference_boundary.md)

A local privacy gateway that removes sensitive values before remote inference, preserves session coherence with typed placeholders, and restores approved values only at a trusted local boundary.
