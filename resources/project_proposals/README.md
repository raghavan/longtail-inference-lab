# Project Proposals

Project proposals describe larger systems directions that are broader than one experiment.

A proposal belongs here while it is being shaped. It moves into `projects/` only after a bounded experiment, measurable question, and completion condition are defined.

## Proposals

### [Privacy Aware Inference Boundary](privacy_aware_inference_boundary.md)

A local privacy gateway that sanitizes sensitive values before remote inference and restores approved values only at a trusted local boundary.
