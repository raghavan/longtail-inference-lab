# Archives

Archives preserve experiments that are complete, paused, or superseded.

Archived work remains part of the laboratory record. Its status must be explicit so a design proposal is never mistaken for measured evidence.

## Archived experiments

The first two experiment specifications were superseded before execution when their strongest ideas converged into one clearer active experiment.

[Open the archived experiments](experiments/README.md)

## Archive rules

1. Complete experiments retain their code, measurements, results, limitations, and conclusion.
2. Paused experiments explain why work stopped and what evidence would justify restarting.
3. Superseded experiments identify the project that replaced them.
4. Specifications with no recorded runs state that clearly.
5. Negative and inconclusive results remain visible because they narrow the search space.
6. The root README continues to surface important archived findings so completed work does not become invisible.
