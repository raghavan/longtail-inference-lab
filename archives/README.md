# Archives

Archives hold experiments that are complete, paused, or superseded.

This folder is intentionally empty at the beginning of the lab reset.

## Archive rules

1. Complete experiments retain their code, measurements, results, limitations, and conclusion.
2. Paused experiments explain why work stopped and what evidence would justify restarting.
3. Superseded experiments link to the project that replaced them.
4. Negative results remain visible because they narrow the long tail inference search space.

The root README should continue to surface important archived findings so completed work does not become invisible.
