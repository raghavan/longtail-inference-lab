# 01 Memory Wiki

**Archive status:** Superseded  
**Archived:** July 12 2026  
**Runs recorded:** None  
**Replacement:** [Terminal Artifact Memory](../../../projects/01_terminal_artifact_memory/README.md)

This experiment specification explored whether a human readable Markdown cache could move recurring questions from frontier inference to reliable local inference.

It was archived before execution because Terminal Artifact Memory provides a more concrete workload, an objective verifier, and a direct way to measure capability growth as verified evidence accumulates.

## Ideas carried forward

1. Human readable and diffable Markdown memory.
2. Retrieval guided local inference.
3. Exact, paraphrased, and genuinely novel questions.
4. Routing regret.
5. Reviewed write back.
6. Wiki duplication, staleness, and retrieval usefulness.
7. The search for the smallest local model that remains useful.
