# Archived Experiments

These specifications were archived on July 12 2026 before any experiment runs were recorded.

They are preserved because they contain ideas that shaped the current research direction. They are not empirical results.

## [01 Memory Wiki](01_memory_wiki/README.md)

**Status:** Superseded

This specification explored a human readable knowledge cache, retrieval guided local inference, routing regret, frontier escalation, and reviewed write back.

The active experiment retains the Markdown memory, retrieval, memory health, and local sufficiency ideas, but grounds them in verified terminal tasks.

## [02 Session Capsule Analysis](02_session_capsule_analysis/README.md)

**Status:** Superseded

This specification explored the size, composition, growth, privacy boundary, and compressibility of coding agent sessions.

The active experiment retains privacy safe artifact extraction, provenance, distribution reporting, and measurement before systems design.

## Replacement

Both specifications converged into [Terminal Artifact Memory](../../projects/01_terminal_artifact_memory/README.md).

The replacement asks one concrete question:

> Can verified artifacts from completed terminal benchmark tasks make a fixed lightweight local model increasingly useful on recurring engineering problems?
