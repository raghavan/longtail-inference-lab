# 02 Session Capsule Analysis

**Archive status:** Superseded  
**Archived:** July 12 2026  
**Runs recorded:** None  
**Replacement:** [Terminal Artifact Memory](../../../projects/01_terminal_artifact_memory/README.md)

This experiment specification explored what coding agent sessions contain, how they grow, how well their transcripts compress, and when portable engine state might be worthwhile.

It was archived before execution because the laboratory now has a clearer use for terminal artifacts: transform verified task evidence into memory and measure whether that memory makes a fixed local model more useful.

## Ideas carried forward

1. Privacy safe artifact extraction.
2. Metrics that avoid committing session text.
3. Artifact provenance and hashed identifiers.
4. Separation of transcript, tool output, instructions, and derived state.
5. Distribution, tail, and compression measurements.
6. Measurement before complex transfer system design.
