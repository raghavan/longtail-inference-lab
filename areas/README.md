# Areas

Areas are ongoing responsibilities that keep the Long Tail Inference Lab useful and trustworthy.

Unlike Projects, an Area does not finish when one experiment ends.

## Lab operations

The lab operations area covers:

1. Experiment quality.
2. Reproducibility.
3. Safety and privacy.
4. Result publishing.
5. Repository maintenance.
6. Clear issue and project status.
