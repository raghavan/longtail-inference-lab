# Projects

Projects are the active, bounded experiments of the Long Tail Inference Lab.

A folder enters Projects only when it has a research question, a measurement plan, and a completion condition. Ideas remain in discussion until they are ready for that commitment.

## Active experiments

### 01 Memory Wiki

Status: Specified

Study whether a markdown knowledge cache can move recurring questions from frontier inference to reliable local inference.

[Open project](01_memory_wiki/README.md)

### 02 Session Capsule Analysis

Status: Specified

Measure the size, composition, growth, and compressibility of real coding agent sessions before designing session transfer machinery.

[Open project](02_session_capsule_analysis/README.md)
